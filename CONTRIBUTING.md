# Contribuindo

Obrigado por se interessar em contribuir para o **Sistema Financeiro Familiar Inteligente**!

## Como Contribuir

1.  **Fork** este repositório.
2.  Crie uma **branch** para sua feature (`git checkout -b feature/nova-funcionalidade`).
3.  Faça suas alterações e **commit** (`git commit -m 'Add: nova funcionalidade'`).
4.  Faça **push** para a branch (`git push origin feature/nova-funcionalidade`).
5.  Abra um **Pull Request**.

## Padrões

-   **Commits:** Use Conventional Commits (ex: `feat:`, `fix:`, `docs:`).
-   **Código:** Mantenha o código limpo e comentado onde necessário.
-   **Documentação:** Atualize a pasta `docs/` se alterar fluxos ou arquitetura.
